# 🚀 Data Automation Runner

This repository serves as an **Automation Runner** designed to periodically synchronize data from various endpoints to a Firebase Realtime Database.

## 📊 Operational Status
[![Runner Status](https://github.com/Khofah567765/kolambabi/actions/workflows/sampingan.yml/badge.svg)](https://github.com/Khofah567765/kolambabi/actions/workflows/sampingan.yml)
- **Sync Interval:** Every 5-15 Minutes (Automated)
- **Engine:** Playwright Headless Browser
- **Database:** Firebase RTDB

## ⚙️ Requirements (Secrets)
Ensure the following **GitHub Actions Secrets** are configured for the system to function correctly:

| Secret | Description |
| :--- | :--- |
| `PAT_TOKEN` | Access token for external modules. |
| `TARGET_URL` | Primary data source URL. |
| `STRIP_ID` | Module identity codes for processing. |
| `FIREBASE_DATABASE_URL` | Target database endpoint. |
| `FIREBASE_SERVICE_ACCOUNT` | Database security authentication. |

## 🛠️ Workflow Logic
1. **Trigger:** The system activates via a routine schedule (`cron`) or manual instruction.
2. **Environment:** Executes modules within an isolated container for stability and consistency.
3. **Processing:** Retrieves raw data, performs cleaning, and standardizes formats.
4. **Sync:** Delivers the final output to the database in real-time.
